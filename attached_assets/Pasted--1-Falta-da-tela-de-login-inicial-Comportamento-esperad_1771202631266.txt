🧮 2. Campos percentuais aceitando apenas inteiros

Isso acontece porque o input provavelmente está configurado como:

keyboardType="numeric"


E/ou usando parseInt().


✔ Correção correta

Trocar para:

keyboardType="decimal-pad"


E no processamento:

parseFloat(valor)

✔ Validação recomendada

Permitir até duas casas:

const regex = /^\d+(\.\d{0,2})?$/;

⚖️ 3. Média ponderada do leito 

A lógica correta é:

📊 Fórmula
Média Elemento =
Σ (% minério no leito × teor do elemento no minério) / 100

✔ Função sugerida
function calcularMedia(leito, elemento) {
  let soma = 0;

  leito.forEach(minerio => {
    soma += minerio.percentual * minerio[elemento];
  });

  return soma / 100;
}

✔ Exibição ideal

Mostrar em tempo real conforme usuário digita.



🚨 4. Validação: Percentual do leito precisa fechar 100%

✔ Regra industrial correta
Somatório percentuais == 100

✔ Implementação
const total = leito.reduce(
  (acc, m) => acc + parseFloat(m.percentual || 0),
  0
);

if (total !== 100) {
  mostrarErro("O total do leito deve ser 100%");
}
